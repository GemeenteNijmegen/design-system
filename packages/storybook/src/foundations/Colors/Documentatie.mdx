# Kleuren

## Contrast
Zorg dat je het minimale kleurcontrast in je project hebt volgens de richtlijnen van [WCAG 2.0 - Web Content Accessibility Guidelines (WCAG 2.2)](https://www.w3.org/TR/WCAG22/#contrast-minimum), om ons te helpen op weg naar die AA-beoordeling. Om er zeker van te zijn dat je kleuren precies goed zijn, bekijk deze [kleurcontrastchecker](https://webaim.org/resources/contrastchecker/).
