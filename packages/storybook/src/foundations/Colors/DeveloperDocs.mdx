# Colors developer documentation
Make sure to check the [color palettes documentation](/?path=/docs/foundations-icons-documentatie--docs) for all color palettes.

## Theming
Colors are available in the default theme `nijmegen-theme`. Some colors have a dark theme counterpart which are available in the dark theme (beta) `nijmegen-theme-dark`.

## Requirements
- If you implement the colors make sure to follow the [WCAG 2.0 - Web Content Accessibility Guidelines (WCAG 2.2)](https://www.w3.org/TR/WCAG22/#contrast-minimum);
- Check contrasts via a [contrastchecker](https://webaim.org/resources/contrastchecker/).


## Html example

</div>
<div className="nijmegen-theme-dark"  style={{backgroundColor: 'var(--nijmegen-color-background-default)', padding: '20px'}}>
    <span style={{color: 'var(--nijmegen-interaction-color)'}}>Interaction color dark theme</span>
</div>
