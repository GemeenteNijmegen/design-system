# Layout
De layout die de gemeente Nijmegen gebruikt is gebasseerd op de [Bootstrap v5.3 layout](https://getbootstrap.com/docs/5.3/layout) met een implementatie van de [NLDS principes voor het gebruik van ruimte](https://nldesignsystem.nl/richtlijnen/stijl/ruimte/).

## Breakpoints
De gemeente Nijmegen gebruikt een layout met de volgende 6 breakpoints:

| Naam        | Class infix | Breakpoint| 
| ----------------- | ----------- | ------------------ | 
| Extra small       |             | < <TokenValue token="nijmegen.breakpoint.sm" /> |          
| Small             | sm          | ≥ <TokenValue token="nijmegen.breakpoint.sm" /> |            
| Medium            | md          | ≥ <TokenValue token="nijmegen.breakpoint.md" />             | 
| Large             | lg          | ≥ <TokenValue token="nijmegen.breakpoint.lg" />            | 
| Extra large       | xl          | ≥ <TokenValue token="nijmegen.breakpoint.xl" />           | 
| Extra extra large | xxl         | ≥ <TokenValue token="nijmegen.breakpoint.xxl" />           | 


## Containers
Om de content uit te lijnen op basis van het device of de viewport van de gebruiker worden containers gebruikt. De gemeente Nijmegen onderscheid drie type containers:

- Een container met een `max-width` op ieder responsive breakpoint (`.nijmegen-container`);
- Een container per breakpoint welke een `width: 100%` (volledige breedte) heeft tot hij het gespecificeerde breakpoint bereikt (bijv `.nijmegen-container-md`);
- Een container die altijd een `width: 100%` (volledige breedte) heeft (`.nijmegen-container-fluid`)

| Naam        | Class infix | `max-width` | padding start | padding end |
| ----------------- | ----------- | ------------------ | ------------------ | ------------------ | 
| Extra small       |             | 100%          | <TokenValue token="nijmegen.container.padding-inline.start" /> |  <TokenValue token="nijmegen.container.padding-inline.end" /> |
| Small             | sm          | 540px             | <TokenValue token="nijmegen.container.padding-inline.start" /> | <TokenValue token="nijmegen.container.padding-inline.end" />|
| Medium            | md          | 720px             | <TokenValue token="nijmegen.container.padding-inline.start" /> | <TokenValue token="nijmegen.container.padding-inline.end" />|
| Large             | lg          | 960px             | <TokenValue token="nijmegen.container.padding-inline.start" /> | <TokenValue token="nijmegen.container.padding-inline.end" />|
| Extra large       | xl          | 	1140px            | <TokenValue token="nijmegen.container.padding-inline.start" /> | <TokenValue token="nijmegen.container.padding-inline.end" />|
| Extra extra large | xxl         | 	1320px            | <TokenValue token="nijmegen.container.padding-inline.start" /> | <TokenValue token="nijmegen.container.padding-inline.end" />|

## Grid
De gemeente Nijmegen gebruikt een 12 column grid binnen 1 row welke anders kan acteren vanaf de verschillende breakpoints. De columns kennen een gap van <TokenValue token="nijmegen.grid.gap" />.
