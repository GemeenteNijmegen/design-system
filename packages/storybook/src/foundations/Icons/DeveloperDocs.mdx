# Icons developer documentation <IconIcons stroke={2} />

Icons that are used are simple and recognizable. Icons always follow a thin line style. All website of the municipality of Nijmegen must use the same functional and alert icons. 

## Icon sets
For the functional and alert icons the [Tabler icons](https://github.com/tabler/tabler-icons) are used. While it is recommended to use the Tabler icons for functional icons, users can choose alternative icons if a suitable option within the Tabler icons is not available.

For task specific icons you can use the [line iconset of OpenGemeente](https://github.com/OpenGemeenten/Iconenset/tree/master/Line). 

## Requirements
- Icons always follow a thin line style;
- Allways use the same functional and alert icons, [see documentation](/?path=/docs/foundations-icons-documentatie--docs);
- Render icons as inline svg;
