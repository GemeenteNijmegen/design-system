# Documentatie link list component

Het Link List-component is een gestructureerde manier om meerdere links overzichtelijk te presenteren in lijstvorm. Deze component is bedoeld om de gebruiker op een toegankelijke en intuïtieve manier door verschillende pagina’s of secties te navigeren.

Het Link List-component biedt een gestructureerde manier om meerdere links te presenteren in een lijstvorm.

Dit component is een Nijmegen uitwerking

## Principes

- Consistentie: Alle links zijn uniform gestyled voor een consistente gebruikerservaring.
- Toegankelijkheid: De component voldoet aan de toegankelijkheidsrichtlijnen (WCAG) door duidelijke focus- en hoverstaten.
- Interactiviteit: Visuele feedback wordt geboden bij interactie met links, zoals hover- en focus-states.
