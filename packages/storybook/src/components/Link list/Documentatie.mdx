# Documentatie link list component

Het Link List-component is een gestructureerde manier om meerdere links overzichtelijk te presenteren in lijstvorm. Deze component is bedoeld om de gebruiker op een toegankelijke en intuïtieve manier door verschillende pagina’s of secties te navigeren.

Het Link List-component biedt een gestructureerde manier om meerdere links te presenteren in een lijstvorm.

## Component

Dit component is een implementatie van het `utrecht-link-list` component.

| Link                                                                                                                                           | Omschrijving                                                    |
| ---------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------- |
| [NPM Utrecht link list css](https://www.npmjs.com/package/@utrecht/link-list-css) OF [NPM Utrecht Components](https://www.npmjs.com/package/@utrecht/components)                                                                    | Het huidige component vereist het `@utrecht/link-list-css` pakket voor een specifieke link list implementatie of het `@utrecht/link-list-css` pakket indien er meerdere componenten geimplementeerd worden. |
| [NPM Utrecht Component Library React](https://www.npmjs.com/package/@utrecht/component-library-react)                                                                    | Voor de React implementatie is dit pakket vereist. |
| [Figma gemeente Nijmegen](https://www.figma.com/design/g3AieWAFD3NtkbDDTTFC30/Novio---Nijmegen-Design-System?node-id=3985-17445) | Design documentatie van dit component.    

## Principes

- Consistentie: Alle links zijn uniform gestyled voor een consistente gebruikerservaring.
- Toegankelijkheid: De component voldoet aan de toegankelijkheidsrichtlijnen (WCAG) door duidelijke focus- en hoverstaten.
- Interactiviteit: Visuele feedback wordt geboden bij interactie met links, zoals hover- en focus-states.
